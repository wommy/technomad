# technomad-jekyll_theme

build by [wommy](https://github.com/wommy)

## Installation

    $ gem install technomad-jekyll_theme

```ruby
Gemfile
---
gem "technomad-jekyll_theme"
```

```yaml
_config.yml
---
theme: technomad-jekyll_theme
```

    $ bundle install
    $ bundle exec jekyll serve
